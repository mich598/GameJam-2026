# The Lemonade Stand
Written by Michael Chen, Jai Lette, Kaleb McKweon (League of Lemons)

## Description
This web-based lemonade stand game is inspired by classic games from our childhood like Papa's Freezeria found on web platforms like Coolmath Games. Drawing inspiration from The Duck Song, the gameplay follows a child's journey to save up enough money to buy a bike by running a friendly neighbourhood lemonade stand. The first person POV introduces an immersive experience and creates greater user engagement and relatability by allowing the player to see the world through the child's eyes. Styled with warm, colourful visuals inspired by Animal Crossing, the game recreates the nostalgic feeling of our childhood summers, having simple goals like buying a bike, and community interactions of selling lemonade while revitalising the once common idea of kids selling lemonade stands in suburban streets. The playful piano tune is designed to help players relax without overpowering the main functionality of the game. 

Designed for older primary school students and aligned with Australian Year 5 curriculum standards, the game combines entertainment with active learning and recall. Players strengthen maths skills through practical gameplay mechanics such as calculating money using decimals, identifying fractions when mixing lemonade, and solving geometric area problems involving rectangles, triangles, and squares. The interactive minigames encourage active recall and hands-on problem solving by targeting geometric reasoning and strengthen cognitive functions such as atttention, concentration and focus. While optional, these minigames directly impact the player’s earnings which means that students who engage with the educational challenges are more likely to save enough money to achieve the final objective of purchasing the bike.

The two minigames have been included as seperate extensions should the player choose to play them individually. Jai_minigame is the geometric reasoning game while Lemon Matching Minigame is the card based memory style matching game. 

## How to run
1. Download the folder Final_Submission and extract to your local disk
2. Open "index.html"
3. Ensure you have the Live Server Extension downloaded for VS code
4. Press "Go live" at the bottom right corner and enjoy
